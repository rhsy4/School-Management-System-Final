## 📖 Тухай

**Нэгдсэн Цахим Сургуулийн Удирдлагын Систем** нь PHP + MySQL технологид суурилсан вэб програм юм. Сургуулийн захиргаа, багш, сурагч, эцэг эхийн өдөр тутмын ажлыг цахимжуулах зорилготой.

### Зорилтот хэрэглэгч
- Дархан-Уул аймгийн 30+ сургууль
- 800–2000 сурагчтай томоохон сургуулиуд
- SaaS хэлбэрээр олон сургуульд нэгэн зэрэг үйлчлэх боломжтой

### 🔐 Нэвтрэлт & Аюулгүй байдал

## 🚀 Суулгах заавар

### Шаардлага
- PHP 8.0 ба түүнээс дээш
- MySQL 5.7 ба түүнээс дээш
- Apache/Nginx вэб сервер
- Composer (сонголттой)

### Алхам 1: Клон хийх
```bash
git clone https://github.com/rhsy4/School-Management-System-Final.git
cd School-Management-System-Final/school_system1
```

### Алхам 2: Тохиргоо
```bash
cp .env.example .env
```
`.env` файлыг нээгээд өөрийн мэдээллээр бөглөнө:
```env
DB_HOST=localhost
DB_NAME=school_db
DB_USER=root
DB_PASS=your_password
APP_URL=http://localhost/school_system1
```

### Алхам 3: Өгөгдлийн сан үүсгэх
```bash
mysql -u root -p < database/schema.sql
```

### Алхам 4: Seed өгөгдөл оруулах (тест)
```bash
php seed.php
```

### Алхам 5: Системд нэвтрэх
```
URL: http://localhost/school_system1
Админ нэвтрэх нэр: admin
Нууц үг: Admin@1234
```

> ⚠️ **Анхааруулга:** Анхны нэвтрэлтийн дараа нууц үгийг заавал солино уу!

## 🔒 Аюулгүй байдал

Энэ систем дараах аюулгүй байдлын арга хэмжээнүүдийг хэрэгжүүлсэн:

- ✅ **SQL Injection** — PDO prepared statements
- ✅ **XSS** — `htmlspecialchars()` бүх гаралтад
- ✅ **CSRF** — token баталгаажуулалт бүх форм дээр
- ✅ **Brute Force** — нэвтрэлтийн оролдлого хязгаарлах
- ✅ **2FA** — Google Authenticator дэмжлэг
- ✅ **Нууц үг** — bcrypt хэшлэлт
- ✅ **Файл байршуулалт** — төрөл, хэмжээ хязгаарлалт
- ✅ **Audit лог** — бүх чухал үйлдлийг бүртгэх
